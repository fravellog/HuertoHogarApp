## Funcionalidades

* **Pantalla de inicio:** Muestra un banner y una lista de productos.
* **Pantalla de detalle de producto:** Muestra los detalles de un producto y permite agregarlo al carrito.
* **Pantalla de carrito:** Muestra los productos agregados al carrito, permite modificar la cantidad y proceder al pago.
* **Pantalla de Blog:** Muestra artículos de un blog.
* **Pantalla de añadir/editar producto:** Permite a los administradores añadir o editar productos.
